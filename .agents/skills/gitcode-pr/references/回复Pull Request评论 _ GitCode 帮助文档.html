回复Pull Request评论

POST 
https://api.gitcode.com/api/v5/repos/:owner/:repo/pulls/:number/discussions/:discussion_id/comments

Request​

Path Parameters

    owner stringrequired

    仓库所属空间地址(组织或个人的地址path)

    repo stringrequired

    仓库路径(path)

    number integerrequired

    第几个PR，即本仓库PR的序数

    discussion_id stringrequired

    讨论id

Query Parameters

    access_token string

    用户授权码

    bodystringrequired
